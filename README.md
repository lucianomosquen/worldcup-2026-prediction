# World Cup 2026 Prediction

Proyecto personal para predecir resultados del Mundial 2026 usando datos históricos de selecciones y modelos de Machine Learning.

## Objetivos

- Reunir un dataset robusto de partidos internacionales.
- Crear métricas para medir el rendimiento actual de cada selección.
- Entrenar modelos para predecir partidos y el torneo completo.
- Dejar el proyecto bien documentado para usarlo como portfolio.
